Появление CSS

CSS — это язык, служащий для описания оформления внешнего вида сайта, созданного с использованием языка разметки (HTML, XHTML, XML). Буквально CSS - Cascading Style Sheets, что означает «каскадные таблицы стилей». Данная технология была создана в декабре 1996 года, популярна среди веб-разработчиков стала с 1998 года, когда вышла вторая версия CSS.
